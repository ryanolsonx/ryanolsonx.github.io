Achieving Zen by using Vim in a minimalist way (while being highly productive)

By Ryan Olson
November 17, 2022

My .vimrc

syntax off
set autoindent
set softtabstop=2
set shiftwidth=2
set expandtab
set backspace=2
set path+=app/javascript/**,src/**
set number
nn <silent> <space> :e %:h<cr>
set backupdir=/tmp
set dir=/tmp

Overview

Over the past several years, I've been fascinated with "getting things done" with less. I believe that a strong software engineer can be productive with even the simplest of tools*. I used to really customize my .vimrc and install dozens of plugins to get all the functionality I thought I need in my Vim text editor.

* See Geohot (a Vim user with a very small .vimrc) or Jonathan Blow (an Emacs user that mostly uses Emacs stock) to see that you can get a lot done without an IDE
