Why isn't this site using CSS?

I find that there are many benefits to using plain ol' HTML:

  • Speed: websites are natively fast. Like insanely fast. It's what we add that slows them down.
  • Simplicity: there's nothing more simple than editing HTML and shipping it.
  • Accessible: you can access a simple web page on the weakest device and slowest network. Easy.
  • Minimalism: I like to keep things simple and minimalistic. I think it's fun making a website like this again.

This site uses straight up HTML on purpose. At work, of course, I use more than just HTML. After all, I specialize in front-end development.
